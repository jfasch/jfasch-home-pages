Wiring¶

  • A0-A5 to ground. Highest bit is hardwired to 1, so this gives 0b1000000, or 0x40

Configure Raspberry¶

/boot/config.txt¶
dtparam=i2c_arm=on
dtoverlay=i2c-pwm-pca9685a,addr=0x40

  • addr=0x40 can be omitted (is default anyway)

  • Reboot

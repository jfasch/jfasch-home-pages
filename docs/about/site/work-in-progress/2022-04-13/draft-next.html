Introduction: Clean Code? What is that?¶

Clean Code: A Handbook of Agile Software Craftsmanship, by Robert C. Martin (“Uncle Bob”)

Cornerstones …

  • Software development cycle (not (!!): waterfall)

  • Requirements

  • Intuitive implementation (⟶ respect from co-workers and oneself)

  • Refactoring: removing unclean code and not breaking requirements

  • Craftsmanship?

  • Maintainability?

Project Structure And Goal¶

Basic Components¶

Create Linux prototype implementations for the hardware devices that are found on a car. These implementations should be usable from many possible architectures, including the reference architectures below, but also from more static architectures.

  • Proximity Sensor: VL53L1X

  • Gyroscope (BNO055)

  • Motor Control: BTN9960LV

  • RC Servo Motor Control

  • PMW Abstraction On Top Of /sys/class/pwm/

What’s to come?

  • devices needing calibration

  • configure acceleration ramps on top of motor control

On top of the basics components, create a reference architecture

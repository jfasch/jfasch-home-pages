Using Gaphor As A Library¶

Install¶

Gaphor comes with a python library to use. python -m pip install gaphor fails because development libraries for the underlying desktop and drawing system are unavailable.

$ sudo dnf install cairo-devel gobject-introspection-devel cairo-gobject-devel

$ python -m pip install gaphor

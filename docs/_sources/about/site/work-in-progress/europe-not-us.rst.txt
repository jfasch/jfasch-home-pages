Migrating Homepage
==================

Headline
--------

.. code-block:: console

   jfasch@laptop:~/My-Projects/jfasch-home$ git remote rename origin github
   jfasch@laptop:~/My-Projects/jfasch-home$ git remote add origin ssh://git@codeberg.org/jfasch/jfasch-home.git

Start
-----

.. code-block:: console

   $ WORKDIR=...

Homepage Main Repo
------------------

* Create repo ``jfasch-home`` on Codeberg
* Redirect ``origin``

  .. code-block:: console

     $ git remote set-url origin ssh://git@codeberg.org/jfasch/jfasch-home.git

* Push to new upstream

  .. code-block:: console

     $ git push

OpenTraining
============

OpenTraining is a set of `Sphinx <https://www.sphinx-doc.org/>`__
extensions which together form a training material management system.

See :doc:`here <netidee/netidee-antrag>` for a video screencast
(German) that briefly describes what it is, and which problems is
solves.

And ``self``?
-------------

.. code-block:: python

   joerg = Person('Joerg', 'Faschingbauer')

* ``Person()``: function call
* *Called* with two parameters
* *Defined* with three parameters
* |longrightarrow| ``self`` added by Python as implicit first
  parameter

  * The object being constructed
  * The *instance* under construction

* ``self`` is just a convention
* Could be ``blah`` just as well

.. code-block:: python

   class Person:
       def __init__(self, firstname, lastname):
           self.firstname = firstname
	   self.lastname = lastname

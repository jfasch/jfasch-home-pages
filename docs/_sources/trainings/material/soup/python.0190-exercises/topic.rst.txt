Exercises
---------

#. In the interactive interpreter, create an empty list. Append to it values of types

   * Integer
   * Floatingpoint
   * Boolean
   * List
   * Tuple
   * Set
   * Dictionary

   Does it work? If yes, print the list using the ``print()``
   *function*.

#. Do the same in an executable Python program
#. What happens when you access a non-existent dictionary member?
#. Write a program that takes a single digit as commandline
   parameter. Print the English word for that digit.

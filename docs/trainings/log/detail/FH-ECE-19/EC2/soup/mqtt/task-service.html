Description¶

  • Turn the standalone publisher program into a systemd service program. Better yet, add a commandline argument that makes the program behave like a systemd service.

    The service will be of type notify (its unit file contains Type=notify) - this means that the program must explicitly notify systemd that they have successfully finished startup an are ready for use.

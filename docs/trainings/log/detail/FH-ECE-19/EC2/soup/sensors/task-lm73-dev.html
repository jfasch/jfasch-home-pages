Temperature Sensor: LM73 (via i2c-dev)¶

  • Description

  • Further Information

  • Dependencies

Description¶

  • Write a Python class that implements the Thermometer interface, and sits on top of the LM73 temperature sensor.

  • Integration: the thermometer class (call it ThermometerLM73_Dev) implements the I2C protocol manually, as described in Linux and I2C (using LM73 Temperature Sensor as Slave): it opens the character device /dev/i2c-1, connects to the device, and writes and reads bytes as shown in the sample code.

    This way, it does not use the kernel driver that comes with the Linux kernel. (The companion task, Temperature Sensor: LM73 (via hwmon), does use the driver.)

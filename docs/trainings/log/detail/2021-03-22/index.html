    Solution, doing everything by hand …

    code/maximum-potschert.py¶
    import sys

left = int(sys.argv[1])
right = int(sys.argv[2])


if left < right:
    print(right)
else:
    print(left)

    Solution, knowing about the map() function and the max() function

    code/maximum-gscheit.py¶
    import sys

print(max(map(int, sys.argv[1:])))

Something Different: Coroutines¶

Another form of parallelism. To get hold of it, a showed (in Python) what generators are, and what asyncio is. The message: both are applications of coroutines.

Generator

Async
fibonacci.py¶
def fibonacci():
    first, second = 1, 1
    yield first
    yield second
    while True:
        third = first + second
        yield third
        first, second = second, third

for num in fibonacci():
    print(num)
asio.py¶
import asyncio

async def one_loop():
    while True:
        await asyncio.sleep(1)
        print('hallo one_loop')

async def another_loop():
    while True:
        await asyncio.sleep(1.1)
        print('hallo another_loop')

async def main():
    async with asyncio.TaskGroup() as tg:
        tg.create_task(one_loop())
        tg.create_task(another_loop())

asyncio.run(main())

In Coroutines: An Overview we see the C++ approach. Neither generators nor async is already there in C++20, although C++23 has generators.

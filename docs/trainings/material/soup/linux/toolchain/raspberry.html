The Raspberry Toolchain¶

Find out whether your Raspberry is running a 32 or 64 bit OS, and install toolchain accordingly 1:

Raspberry

Host/PC (as root)

 
$ uname -m
armv7l

Debian

# apt install gcc-arm-linux-gnueabihf

Fedora

# dnf install gcc-arm-linux-gnu

$ uname -m
aarch64

# apt install gcc-aarch64-linux-gnu

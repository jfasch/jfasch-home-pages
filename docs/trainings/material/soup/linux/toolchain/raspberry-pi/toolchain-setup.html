Download And Install Toolchain Archive¶

See also

  • Building A Toolchain On Fedora For Ubuntu 18.04 (WSL)

Download:

  • armv8-rpi4-linux-gnueabihf.tar.xz

  • Download the toolchain archive (a whopping 70MB), and unpack to <toolchain-dir>

  • Modify PATH (best permanently in ~/.bashrc). Make sure that <toolchain-dir> is given as absolute path (see here).

$ export PATH=<toolchain-dir>/armv8-rpi4-linux-gnueabihf/bin:$PATH

Establish CMake Toolchain File¶

Documentation

  • Cross Development

CMake Toolchain File

  • toolchain file

  • Download (best into <toolchain-dir>) toolchain file

  • Open it with a text editor, and modify the folling lines to point to your toolchain

    set(CMAKE_FIND_ROOT_PATH <toolchain-dir>/armv8-rpi4-linux-gnueabihf/armv8-rpi4-linux-gnueabihf/sysroot)
set(CMAKE_SYSROOT <toolchain-dir>/armv8-rpi4-linux-gnueabihf/armv8-rpi4-linux-gnueabihf/sysroot)

  • Build your project with it (Cross Development)

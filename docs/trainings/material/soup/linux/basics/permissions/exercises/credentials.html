Exercise: Determine Your User Credentials¶

Documentation

  • man -s 1 id

Topics

  • Login? What Is That?

  • Use the id command to find out about your user credentials.

  • In particular, write down

    • User name

    • User ID

    • Names of groups you are member of

    • IDs of groups you are member of

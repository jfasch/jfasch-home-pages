Exercise: Remove Directory Tree¶

  • In Exercise: Copy Directory Tree, you created a directory ~/parent-copy,

    $ tree ~/parent-copy/
/home/<you>/parent-copy/
├── child
│   ├── business-card
│   ├── garbage1.txt
│   └── garbage2.txt
├── garbage1.txt
├── garbage2.txt
└── hello

  • Remove that directory entirely, so that,

    $ ls ~/parent-copy
ls: cannot access '/home/jfasch/parent-copy': No such file or directory

Setup And Usage#

Documentation

  • ENABLE_TESTING()

  • Enable testing (ideally in toplevel CMakeLists.txt)

    ENABLE_TESTING()

  • This creates a test target

    $ cmake /home/jfasch/work/jfasch-home/trainings/material/soup/cmake/14-testing/
$ make test
Running tests...
Test project /home/jfasch/tmp/cmake-demo
No tests were found!!!

Add Simple Test#

Documentation

  • ADD_TEST()

  • ADD_TEST() adds a program to run when test target is run

  • Exit status 0 ‚ü∂ Ok

  • Exit status not 0 ‚ü∂ Failure

bin/CMakeLists.txt#
ADD_TEST(
  NAME hello-first-runs
  COMMAND hello-first)

$ make test
Running tests...
Test project /home/jfasch/tmp/cmake-demo
    Start 1: hello-first-runs
1/1 Test #1: hello-first-runs .................   Passed    0.00 sec

100% tests passed, 0 tests failed out of 1

Total Test time (real) =   0.01 sec

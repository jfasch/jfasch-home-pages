My First Program (1)¶

hello.c¶
#include <stdio.h>

int main(void)
{
    printf("hello, world\n");
    return 0;
}
Build and execute¶
$ gcc hello.c
$ ./a.out
hello, world

What we see …

  • A program consists of functions and variables

  • Functions consist of statements

  • Function call (printf()) is a statement

  • main() is special (int return value ⟶ OS Exit Status)

  • Building appears simple but isn’t

  • stdio.h??

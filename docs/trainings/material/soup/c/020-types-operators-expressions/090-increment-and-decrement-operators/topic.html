Confused to perfection (1)¶

No confusion: copying string while ignoring all occurences of character c …

void copy_and_omit(char dst[], const char src[], char c)
{
    int i = 0, j = 0;

    while (src[i] != '\0') {
        if (src[i] != c) {
            dst[j] = src[i];
            j = j + 1;
        }
        i = i + 1;
    }
    dst[j] = '\0';
}

Confused to perfection (2)¶

Real men complain:

  • So many lines for a trivial thing?

  • Multiple indexing does not perform!

void copy_and_omit(char dst[], const char src[], char c)
{
    int i = 0, j = 0;
    char cur;

    while ((cur = src[i++]) != '\0')
        if (cur != c)
            dst[j++] = cur;
    dst[j] = '\0';
}

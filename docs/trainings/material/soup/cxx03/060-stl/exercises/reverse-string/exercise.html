Exercise: Reverse a String¶

  • Description

  • Dependencies

Description¶

Write a program that

  • reads a string from standard input 1

  • reverses that string 2

  • outputs the reversed string to standard output 3

$ ./reverse-string
abc               <--- user input, typed by you
cba               <--- program output

Footnotes

1

Use this code snippet to read a string from standard input:

std::string s;
std::cin >> s;
2

Iterating over a string (here in forward iteration) is done just like you would iterate over an array.

std::string s("abc");
for (int i=0; i<s.size(); i++) {
    char c = s[i];
    ...
}
3

Single characters are appended to a string like so:

std::string s("cb");
s += 'a';

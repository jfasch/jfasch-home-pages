Screenplay: Removing (Erasing) Elements¶

  • By Pointer (Err, Iterator)

  • By Key

By Pointer (Err, Iterator)¶

  • Have to have an iterator pointing to to-be-removed element

  • Usually by some find operation

code/map-erase-iter.cpp¶
#include <map>
#include <string>
#include <iostream>

using MyMap = std::map<int, std::string>;

int main()
{
    MyMap my_map = {
        { 2, "zwei" },
        { 51, "einundfuenfzig" },
        { 54, "vierundfuenfzig" },
        { 346, "dreihundertsechsundvierzig" },
        { 1001, "tausendeins" },
        { 1002, "tausendzwei" },
    };

    auto elem = my_map.find(1001);   // <--- points to valid tree element
    my_map.erase(elem);              // <--- removes that element

    std::cout << my_map.size() << std::endl;

    return 0;
}

By Key¶

code/map-erase-key.cpp¶
#include <map>
#include <string>
#include <iostream>

using MyMap = std::map<int, std::string>;

int main()
{
    MyMap my_map = {
        { 2, "zwei" },
        { 51, "einundfuenfzig" },
        { 54, "vierundfuenfzig" },
        { 346, "dreihundertsechsundvierzig" },
        { 1001, "tausendeins" },
        { 1002, "tausendzwei" },
    };

    size_t n_removed = my_map.erase(1001);
    std::cout << "Number of elements removed: " << n_removed << std::endl;
    std::cout << "Numer of elements remaining: " << my_map.size() << std::endl;

    return 0;
}

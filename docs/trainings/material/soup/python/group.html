Python Programming: From Absolute Beginner to Advanced Productivity¶

Duration

  • Modules take a maximum of three days (most are shorter)

  • Modules can be booked separately

Course Language

  • Course material language is English

  • Trainer language is English or German

Booking

training@faschingbauer.co.at

Python Programming is a series of courses, each about two or three days worth of content (mixed lecture and exercises).

Course Material¶

  • Python: The Language Fundamentals
  • Python: More Language Features
  • Python: Project/Package Management
  • Python: Miscellaneous Topics
  • Python: Draft Topics

Sample Trainings¶

Customers are usually left with post-mortem course material - a transcript of what has happened, relevant links, and more. Here is a representative list of such transcripts,

  • Python Basic and Advanced (22.3.2021 - 25.3.2021). Four days of Python, gradually advancing from basic to advanced topics.

  • Python Individualtraining (18.5.2020 - 19.5.2020). Single-person training in an online setting.

  • Python Basic and Advanced (19.10.2020 - 22.10.2020). Four days of Python, with a number of excursions into the customer’s domain, like CAN Bus communication

  • Python Advanced (7.9.2020 - 10.9.2020). Very advanced; accompanied by a lot of improvised live-hacking.

  • Python (12.11.2019 - 14.11.2019 in Graz). Fun beating MATLAB with NumPy, for a team who wanted to know if that is possible.

Boolean Expressions¶

[14]:

flt = (df['firstname'] == 'Joerg') & (df['lastname'] == 'Faschingbauer') | (df['firstname'] == 'Philipp')

Attention: braces are important because ‘&’ binds stronger than ‘==’. This is bad.

[15]:

df[flt]

Exercise: Refactoring - Extract Both CSV Formats Into Module¶

  • Requirement

  • Dependencies

Requirement¶

Create a module userdb_csv.py to be imported from whatever program. That module will contain

  • The function you created in Exercise: Refactoring - Extract CSV Reading Into Function; that is the function which reads our user database from a CSV file which has a header line.

  • A function (to be created likewise) that reads our user database from header-less CSV file. Extract that function from the solution to Exercise: Read CSV File (csv.reader).

  • Modify both programs (read-userdb.py and read-userdb-header.py) to import the module, and to use the functions from there.

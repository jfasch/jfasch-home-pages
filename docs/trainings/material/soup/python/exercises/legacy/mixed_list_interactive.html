Exercise: Mixed List (Interactive Interpreter)¶

In the interactive interpreter, create an empty list. Append to it values of types

  • Integer

  • Floatingpoint

  • Boolean

  • String

  • List

  • Tuple

  • Set

  • Dictionary

Does it work? If yes, print the list using the print() function.

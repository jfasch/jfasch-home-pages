Exercise: Boolean Expressions¶

  1. What is the value of the following expression?

(value1 > 500) or (value2 < 800) and not (value3 == 400)

Assuming that the variables used have the following values:

    • value1 = 501

    • value2 = 799

    • value3 = 400

    • value1 = 500

    • value2 = 800

    • value3 = 0

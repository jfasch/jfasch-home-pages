Groundwork¶

In the exercises Exercise: Refactoring - Extract CSV Reading Into Function (csv.reader) and Exercise: Refactoring - Extract CSV Reading Into Function (csv.dictreader), we created two functions that returned lists of user records. A user record is a dictionary of predefined form:

{
   'id': int(id),
   'firstname': firstname,
   'lastname': lastname,
   'birth': birth,
}

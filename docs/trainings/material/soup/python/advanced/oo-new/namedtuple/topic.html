Shortcut For Simple Data-Holding Classes¶

class Person:
    def __init__(self, firstname, lastname):
        self.firstname = firstname
        self.lastname = lastname

… is the same as …

named-tuple-simple.py¶
from collections import namedtuple

Person = namedtuple('Person', ('firstname', 'lastname'))

person = Person('Joerg', 'Faschingbauer')
print('Firstname:', person.firstname)
print('Lastname:', person.lastname)

$ python named-tuple-simple.py
Firstname: Joerg
Lastname: Faschingbauer

Constructing From Iterable: _make()¶

  • Data-holding classes are popular with tabular data (CSV, databases, …)

  • Rows are iterable

  • ⟶ someclass._make()

named-tuple-csv.py¶
from collections import namedtuple

persons_from_csv = [
    ['Joerg', 'Faschingbauer'],
    ['Caro', 'Faschingbauer'],
]

Person = namedtuple('Person', ('firstname', 'lastname'))
for person in map(Person._make, persons_from_csv):
    print('Firstname:', person.firstname)
    print('Lastname:', person.lastname)

$ python named-tuple-csv.py
Firstname: Joerg
Lastname: Faschingbauer
Firstname: Caro
Lastname: Faschingbauer

Convert To dict: _asdict()¶

named-tuple-asdict.py¶
from collections import namedtuple

Person = namedtuple('Person', ('firstname', 'lastname'))

person = Person('Joerg', 'Faschingbauer')
person_dict = person._asdict()
print(person_dict)

$ python named-tuple-asdict.py
{'firstname': 'Joerg', 'lastname': 'Faschingbauer'}

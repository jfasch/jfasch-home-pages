Building class rectangle Upon class point#

In the project you have set up in Exercise (FH): class point, you’ll find an a test suite for a class point3d, together with an empty point3d.{h,cpp} combo.

Continuing the tradition of test driven development, continually extend class point3d by adding the following tests one by one.

Default Constructor#

Topics

  • Default Constructor

Use the C++11 = default and initialize the members at their definition. (See here)

code/tests/point3d-default_ctor.cpp#
#include <gtest/gtest.h>
#include <point3d.h>

TEST(point3d_suite, default_ctor)
{
    const point3d p;

    ASSERT_EQ(p.x(), 0);
    ASSERT_EQ(p.y(), 0);
    ASSERT_EQ(p.z(), 0);
}

(In-)Equality#

  • Do not pass the right hand side operand by copy [1].

code/tests/point3d-operator_eq_ne.cpp#
#include <gtest/gtest.h>
#include <point3d.h>

TEST(point3d_suite, operator_eq_ne)
{
    const point3d p{1,2,3};
    const point3d q{3,4,5};

    bool b;

    b = (p == q);
    ASSERT_FALSE(b);

    b = (p != q);
    ASSERT_TRUE(b);

    b = (p == p);
    ASSERT_TRUE(b);

    b = (p != p);
    ASSERT_FALSE(b);
}

+=#

  • Implement that one in the point3d.cpp file, please.

  • Do not pass the right hand side operand by copy [1].

code/tests/point3d-operator_pluseq.cpp#
#include <gtest/gtest.h>
#include <point3d.h>

TEST(point3d_suite, operator_pluseq)
{
    point3d p{1,2,3};
    const point3d vec{3,4,5};
    const point3d& p1 = p += vec;

    ASSERT_EQ(p.x(), 4);
    ASSERT_EQ(p.y(), 6);
    ASSERT_EQ(p.z(), 8);

    bool b = (p == p1);
    ASSERT_TRUE(b);

    ASSERT_EQ(&p1, &p);  // operator+=() returns point3d& !!
}

Absolute Value#

Documentation

  • sqrt()

  • Implement that one in the point3d.cpp file, please.

  • Do not pass the right hand side operand by copy [1].

  • Use the std::sqrt() function from <cmath> to compute a square root.

code/tests/point3d-abs.cpp#
#include <gtest/gtest.h>
#include <point3d.h>

TEST(point3d_suite, abs)
{
    const point3d p{3,4,5};
    double abs = p.abs();

    ASSERT_FLOAT_EQ(abs, 7.071067812);
}

Distance Between Two Points#

code/tests/point3d-distance.cpp#
#include <gtest/gtest.h>
#include <point3d.h>

TEST(point3d_suite, distance)
{
    const point3d p1{3,4,5};
    const point3d p2{5,7,8};
    double dst = p1.distance(p2);

    ASSERT_FLOAT_EQ(dst, 4.69041576);
}
